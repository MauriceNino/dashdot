# Integration

To integrate dash. in your website, you can simply embed an iframe, like in the following example

```html
<iframe
  src="http://localhost:3000/?singleGraphMode=true&graph=cpu&multiView=true"
  style="border-radius: 20px"
>
</iframe>
```
