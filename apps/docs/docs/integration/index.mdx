# Integrations

## Homarr

### Preview

<img
  src='/img/homarr_integration.png'
  alt='Homarr screenshot with dash. integration'
  style={{ maxHeight: '350px' }}
/>

### Setup

Website: <https://homarr.dev>

1. Install and setup Homarr [according to the docs](https://homarr.dev/docs/introduction/installation)
2. Enable the Dashdot integration in the Homarr settings
3. Add your Dashdot instance to your services
4. You can then customize your widget directly in Homarr

## Heimdall

### Preview

<img
  src='/img/heimdall_integration.png'
  alt='Heimdall screenshot with dash. integration'
  style={{ maxHeight: '350px' }}
/>

### Setup

Website: <https://heimdall.site/>

1. Install and setup Heimdall [according to the docs](https://github.com/linuxserver/Heimdall/discussions/778)
2. Add a new service from the application list
3. Click on "Application Type" and select `Dashdot`
4. Optionally: Change the title to "Dash."
