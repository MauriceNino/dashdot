  src='/img/homarr_integration.png'

/>

### Setup

Website: [https://homarr.dev](https://homarr.dev)

1. Install and setup Homarr [according to the docs](https://homarr.dev/docs/getting-started/)
2. Enter the edit mode
3. Add the Dash. widget from the header menu
4. Drag your widget to your desired location on the dashboard
5. Click the three dots and edit the `Dash. URL` field to your Dash. instance.
6. Click save and exit the edit mode

### Setup

Website: [https://heimdall.site](https://heimdall.site)

1. Install and setup Heimdall [according to the docs](https://github.com/linuxserver/Heimdall/discussions/778)
2. Add a new service from the application list
3. Click on "Application Type" and select `Dashdot`
4. Optionally: Change the title to "Dash."
