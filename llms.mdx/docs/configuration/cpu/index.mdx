# CPU







Options [#options]

DASHDOT_CPU_LABEL_LIST [#dashdot_cpu_label_list]

Selects which data is shown in the CPU widget.

The available options are: `brand`, `model`, `cores`, `threads`, `frequency`.

* type: `string (comma separated list)`
* default: `brand,model,cores,threads,frequency`

DASHDOT_ENABLE_CPU_TEMPS [#dashdot_enable_cpu_temps]

If you want to show the CPU temperature in the graph.
This will probably not work on a VPS, so you need to try it on your own if this works.
For home servers it might work just fine.

* type: `boolean`
* default: `false`

</Callout>

DASHDOT_CPU_TEMPS_MODE [#dashdot_cpu_temps_mode]

To choose between average and max CPU temperature (accross all cores). Needs [`DASHDOT_ENABLE_CPU_TEMPS`](./cpu#dashdot_enable_cpu_temps) enabled to work.

The available options are: `avg`, `max`.

* type: `string`
* default: `avg`

DASHDOT_CPU_CORES_TOGGLE_MODE [#dashdot_cpu_cores_toggle_mode]

Switches the Processor core view depending on the selected option. The `toggle`
option allows you to switch the view from the dashboard, other options hide the
toggle from the dashboard.

The available options are: `toggle`, `multi-core`, `average`.

* type: `string`
* default: `toggle`

Styles [#styles]

DASHDOT_CPU_WIDGET_GROW [#dashdot_cpu_widget_grow]

To adjust the relative size of the Processor widget.

* type: `number`
* default: `4`

DASHDOT_CPU_WIDGET_MIN_WIDTH [#dashdot_cpu_widget_min_width]

To adjust the minimum width of the Processor widget (in px).

* type: `number`
* default: `500`

DASHDOT_CPU_SHOWN_DATAPOINTS [#dashdot_cpu_shown_datapoints]

The amount of datapoints in the Processor graph.

* type: `number`
* default: `20`

DASHDOT_CPU_POLL_INTERVAL [#dashdot_cpu_poll_interval]

Read the Processor load every x milliseconds.

* type: `number`
* default: `1000`

Overrides [#overrides]

You can use overrides to manually set statically gathered data. This is useful
if you want to use dashdot on a VPS or a system where the data cannot be
gathered automatically.

DASHDOT_OVERRIDE_CPU_BRAND [#dashdot_override_cpu_brand]

* type: `string`
* default: `unset`

DASHDOT_OVERRIDE_CPU_MODEL [#dashdot_override_cpu_model]

* type: `string`
* default: `unset`

DASHDOT_OVERRIDE_CPU_CORES [#dashdot_override_cpu_cores]

* type: `number`
* default: `unset`

DASHDOT_OVERRIDE_CPU_THREADS [#dashdot_override_cpu_threads]

* type: `number`
* default: `unset`

DASHDOT_OVERRIDE_CPU_FREQUENCY [#dashdot_override_cpu_frequency]

Number needs to be passed in GHz (e.g. `2.8`)

* type: `number`
* default: `unset`
